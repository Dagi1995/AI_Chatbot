# AI_Chatbot

A modern AI-powered chatbot built with **Next.js**, **React**, and **Tailwind CSS**, featuring markdown parsing, LaTeX support, and an interactive UI.

---

## Features

- **Real-time AI responses** using `@ai-sdk/react` and `ai` packages.
- **Markdown rendering** with full support for:
  - Bold, italics, strikethrough
  - Headings, lists, tables
  - Code blocks with copy functionality
  - LaTeX math formulas (`rehype-katex` & `remark-math`)
- **Embla Carousel** integration for scrollable components.
- **Keyboard navigation** support for carousel items.
- **Secure rendering** with `harden-react-markdown` to prevent incomplete markdown rendering.
- **Optimized for Next.js** with Tailwind CSS styling.
